Klipper firmware CREALITY-F401.bin
Version: v0.13.0-693-g5ac7c301 (built 2026-07-17, Yumi-Lab fork, post-build verified)

TARGET: Creality STM32F401 board (Ender-5 S1 / Ender-3 S1 Pro / CR-10 Smart Pro,
USART1 PA10/PA9 @250000, 64KiB bootloader).
FLASH: on a FAT32 SD card create a folder named STM32F4_UPDATE, copy the file
INSIDE it with a unique name ending .bin, insert, power-cycle, wait ~30s.
Older builds are archived in the dated subfolders.
